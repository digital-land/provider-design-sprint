The endpoint URL is the URL of the data file or feed itself. This is where we will regularly collect the data from.

Endpoints typically fall into 1 of the following 2 categories:

  • hosted file — these will usually be URLs which end in something like .json or .csv
  • standards compliant web server — these will usually be identifiable by parts of the URL like MapServer or FeatureServer (types of APIs), or sections that look like query parameters, like ?service=WFS&version=1.0.0

The webpage URL is the URL of the page on your website which contains:

  • information about the dataset
  • the link to your data
